# dev_blog_vagrant

Vagrant template for writing to our dev.solita.fi blog

Following setup:

* CentOS 6.5
* Ruby (stable)
* Gems are installed with bundler 

## Prerequisites

You will need [Virtualbox][1] and [Vagrant][2]

## Running

To start the box run:

	vagrant up
